# 🎲 Dice Game

A simple Dice Game built using **HTML, CSS, and JavaScript**.  
Refresh the page or click the button to roll the dice and see which player wins.

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (DOM + Random Numbers)

---

## 🎮 Features
- Random dice roll for Player 1 and Player 2
- Automatic winner detection
- Refresh-based gameplay or button click (if added)
- Simple and clean UI

## 🎲 How It Works
- JavaScript generates two random numbers (1–6)
- Each number is linked to a dice image
- Images are updated dynamically on the page
- Higher number wins the game
- If both are equal → Draw
